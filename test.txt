🔊 PHONEME AWARENESS (Arabic Speaker Challenges)
═══════════════════════════════════════════════════════════════

⚠️ This table is to help you LISTEN CAREFULLY. NOT to assume errors!
ONLY report errors you ACTUALLY HEAR.

┌──────────────┬──────────────┬──────────────────────────────────┐
│ Phoneme      │ May Sound    │ Example                          │
├──────────────┼──────────────┼──────────────────────────────────┤
│ /θ/ (TH)     │ /s/ or /t/   │ "think" → "sink" or "tink"       │
│ /ð/ (TH)     │ /z/ or /d/   │ "this" → "zis" or "dis"          │
│ /p/          │ /b/          │ "park" → "bark"                  │
│ /v/          │ /f/          │ "very" → "ferry"                 │
│ /ɪ/ vs /iː/  │ confused     │ "ship" vs "sheep"                │
│ /ʌ/ (cup)    │ /a/ or /o/   │ "cup" → "cop"                    │
│ Consonant clusters │ added vowel │ "street" → "is-treet"       │
└──────────────┴──────────────┴──────────────────────────────────┘

AMERICAN ENGLISH SOUNDS:
• Flap T: "water" = /ˈwɑːɾər/ (T sounds like soft D)
• Rhotic R: Always pronounce R: "car" = /kɑːr/
• Dark L: "full" = /fʊɫ/

MINIMAL PAIRS FOR PRACTICE:
• θ→s: think/sink, thick/sick, path/pass, math/mass
• ð→z: this/zis, breathe/breeze, that/zat
• p→b: park/bark, pat/bat, pin/bin, cap/cab
• v→f: very/ferry, van/fan, vine/fine
• ɪ→iː: ship/sheep, bit/beat, sit/seat, live/leave
